# Banking Data Analysis Project


- **What:** Analysis of bank customer data to surface retention and product-insight opportunities and an interactive dashboard.
- **Key skills:** EDA, data cleaning, visualization, dashboarding (Power BI), basic predictive modeling.
- **Files:** `Banking.csv` (data), `Bank.ipynb` (analysis notebook), `Banking Dashboard.pbix` (Power BI dashboard).
